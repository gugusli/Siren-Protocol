# AFTERSHOCK - Testing Checklist

## Funcional
- [ ] Todos los personajes funcionan
- [ ] Todas las mejoras funcionan
- [ ] Todas las sinergias se activan correctamente
- [ ] Eventos especiales se disparan
- [ ] Guardado/carga sin errores
- [ ] Transiciones de escena suaves

## Rendimiento
- [ ] 60 FPS en mid-range con 50+ enemigos
- [ ] Sin memory leaks después de 10 runs
- [ ] Batería dura >3 horas (móvil)
- [ ] Tamaño APK <150MB

## UI/UX
- [ ] Todos los botones responden
- [ ] Textos legibles en pantallas pequeñas
- [ ] No hay overlapping de UI
- [ ] Tooltips informativos

## Balance
- [ ] 5 runs con cada personaje
- [ ] Testear 10 builds diferentes
- [ ] Verificar que no hay builds rotos
